# Blog-IB

Um projeto de blog desenvolvido para partilha de conteúdos, artigos e informações de forma simples e organizada.

##  Sobre o Projeto

O **Blog-IB** é uma plataforma de blog criada com foco em simplicidade, desempenho e boa experiência para o utilizador.  
O sistema permite criar, visualizar e organizar publicações de maneira intuitiva.

##  Funcionalidades

- Criação de posts
- Visualização de artigos
- Sistema de categorias
- Interface responsiva
- Organização de conteúdos
- Estrutura simples e fácil de manter

##  Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- PHP
- MySQL
